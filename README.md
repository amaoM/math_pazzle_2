### 貪欲法（No.5）

> 問題に合わせて作成した基準を使って、簡単に最もよいものを求める方法

ある値段に必要な貨幣の枚数を求める場合、金額の一番高い貨幣から順に必要枚数を数えていく。条件によっては最適解が求まらない場合がある。

別名：グリーディ算法、欲張り方
